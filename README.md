# AKTU CGPA Calculator

A simple and accurate CGPA calculator for AKTU students.

It eliminates the need for manual weighted calculations by letting you input semester-wise SGPA and credits, and instantly computing your CGPA and equivalent percentage.

---

## Overview

AKTU’s CGPA calculation involves weighted averages across semesters, which many students either misunderstand or calculate incorrectly.

This tool provides a straightforward and reliable way to get accurate results without the hassle.

---

## Tech Stack

- React
- Tailwind CSS
- JavaScript

---

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/aanshiksharma/aktu-cgpa-calculator.git
cd aktu-cgpa-calculator
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```
